# ArcRaiders Editor

## Présentation
ArcRaiders Editor est une application web permettant de visualiser, éditer et gérer les données du jeu ARC Raiders (items, quêtes, modules, etc.) au format JSON. Cet outil facilite la contribution et la maintenance des données du projet ArcRaiders Data.

## Installation
1. Cloner le dépôt :
	```bash
	git clone https://github.com/anisselbd/arcraiders-data.git
	```
2. Aller dans le dossier de l’éditeur :
	```bash
	cd arcraiders-editor
	```
3. Installer les dépendances :
	```bash
	npm install
	```
4. Lancer l’application :
	```bash
	npm start
	```
L’éditeur sera accessible sur http://localhost:3000

## Fonctionnalités
- Visualisation et édition des fichiers JSON du projet ArcRaiders Data
- Interface moderne et intuitive
- Recherche et filtrage des données
- Prévisualisation des images associées
